Trabalho feito por Rafael Carneiro Pregardier.
Features implementadas:

- Classes em C++ para definição de vetores.

- Sistema de controle do canhão.

- Sistema de movimentação do projétil.

- Colisão do projétil com os alvos.

- Controle de FPS.

- Explosão dos blocos com partículas.

- Tela de pause.

- Menu inicial.

- Fases do jogo, a cada fase aumenta-se a dificuldade.

- Sons no jogo.
